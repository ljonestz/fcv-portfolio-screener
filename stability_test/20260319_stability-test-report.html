4 · Key findings

Summary and implications for methodology use

  • Pass Composite stability confirmed. Both composites cleared the SD ≤ 0.5 threshold comfortably (max SD = 0.25). The methodology is reliable for portfolio-level classification at the floor of the scoring range.
  • Pass Zero categorical changes across all runs. No run produced a different rating (Not Addressed / Partially Addressed / Substantially Addressed) or a different gap matrix cell. A single screening run is sufficient to classify low-scoring projects.
  • Note D6 and D7 warrant rubric tightening. Implementation Flexibility (SD 0.63) and Results Framework (SD 0.52) show marginally elevated variance. Both reflect genuine interpretive ambiguity in this document type — the boundary between standard project management and FCV-adapted design is not always explicit in 2015-era PADs. Calibration guidance for these two dimensions is recommended before formal cross-country comparisons.
  • Note RF2 flip is analytically meaningful, not noise. The single RF2 trigger (run 7 only) reflects a real ambiguity: the PAD documents distributional impacts through safeguards instruments without applying a conflict-sensitivity framing. The 10% flip rate is at — not above — the stability threshold. No corrective action required; annotate in methodology documentation.
  • Next Boundary-case test pending. P166220 (Djibouti Integrated Cash Transfer, S=5.83, R=5.50) is the critical next test. Both composites sit directly at the gap matrix classification thresholds; a composite SD of ≥ 0.5 could produce quadrant flips. Results will determine whether the methodology requires score-averaging across repeated runs near thresholds.
